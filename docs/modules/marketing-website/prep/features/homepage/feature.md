## Summary

The homepage is the primary entry point for all users. It communicates NicheLearner's core value proposition (personalized assessment, cross-topic references, adaptive guidance) in language students and parents understand, displays credibility markers to build trust, and provides clear calls-to-action to start a free trial.

## User Story

As a student, parent, or educator visiting the website, I want to quickly understand what NicheLearner does and whether it's worth trying, so I can decide whether to sign up for a free trial.

## Acceptance Criteria

- **Value proposition clearly visible above the fold** — the three core capabilities (personalized assessment, cross-topic references, adaptive guidance) are prominently displayed in simple language
- **Student-specific messaging is distinct** — messaging addresses student concerns: honest feedback on preparation level, identification of knowledge gaps, personalized next-step roadmap
- **Parent-specific messaging is distinct** — messaging addresses parent concerns: credibility of platform, outcomes legitimacy, investment justification, how to monitor child's progress
- **Credibility markers are present** — security/privacy stance, how the system works, and who built it are briefly explained
- **Primary CTA is prominent** — "Start Free Trial" or equivalent button is visible and easy to find
- **College partnerships link is present** — secondary navigation or banner for college administrators to access institutional offering information
- **No detailed learning content is listed** — website does not enumerate course subjects, topics, or specific learning materials
- **Page is mobile-friendly** — layout adapts to mobile screen sizes (tested separately in mobile-responsive-design feature)

## Priority

**High** — Homepage is the entry point for all acquisition channels (organic search, affiliate, college referrals); without a credible homepage, users will not convert to trial signup.

## Dependencies

- None (can be developed independently)
